### Integration Branch

 * Integrates only changes from other branches.
 * No direct development on this branch.
 * Contains only **merges**.
 * **1st-Parent-History** shows the sequence of integrations<BR/>
   e.g., "Feature 1", "Bugfix 1", "Feature 2"
