### Feature vs. Trunk

 * **Feature Branching** \
   Features werden auf Branches separat entwickelt und
   erst bei Fertigstellung integriert
 * **Trunk Based Development** \
   Es wird auf einem Branch entwickelt. 
   Änderungen werden unmittelbar integriert.
   Oft werden Feautues
