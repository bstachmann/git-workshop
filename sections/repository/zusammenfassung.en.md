# Repository Summary

---

## Key Concepts

 * A repository is a version-controlled directory.
 * Commits are snapshots of the project.
 * Branches are pointers to commits.
 * Tags are named pointers to specific commits.
 * Remotes link to other repositories.

---

## Commands

```bash
    git init
    git clone <url>
    git remote -v
    git branch
    git tag
```
