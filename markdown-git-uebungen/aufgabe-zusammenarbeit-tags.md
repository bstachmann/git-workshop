# Übung - Tags


## Tipps

* `git tag <name> <revision>` 
*  Tags werden beim *Push* nicht automatisch Übertragen.
   Man gibt entweder einzelne Tags an, oder `--tags` um alle Tags zu übertragen.  
    `git push origin v1.0 v1.1`
    `git push --tags`
